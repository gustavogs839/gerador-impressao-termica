# Gerador de impressão térmica - Elgin i9

Projeto simples em **HTML + CSS + JavaScript** para montar e imprimir modelos para impressora térmica, seguindo o estilo dos exemplos enviados.

## Modelos incluídos

1. `Empadão Goiano`
2. `Empadão Moda`
3. `Empadão Carne de Sol`
4. `Pedido numerado` com número editável
5. `Divulgação do estabelecimento` com QR Code para site
6. `Corretor Imobiliário` com QR Code que abre o WhatsApp com mensagem pronta
7. `Modelo em branco personalizável`

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Escolha o modelo desejado.
3. Edite os campos de texto.
4. Clique em **Imprimir** para enviar para a impressora.
5. Se quiser gerar um arquivo separado, clique em **Baixar HTML**.

## Observações para a Elgin i9

- O layout foi pensado para bobina de **80 mm**, com opção de ajuste para **58 mm**.
- Antes de imprimir em produção, confirme no driver da impressora a largura correta do papel.
- Para melhor resultado, use escala 100% e remova margens extras no diálogo de impressão.
